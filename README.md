NestJS 기반 음성 채널 자동 관리 및 사용자 활동 추적 봇

---

## 🛠 Tech Stack

- **Backend**: NestJS, TypeORM
- **Database**: PostgreSQL
- **Deployment**: Docker, Docker Compose

---

## 📌 Features

### 🔁 Channel Automation
- 사용자 진입 시 음성 채널 자동 생성
- 무인 채널 자동 제거
- 웹 인터페이스를 통한 동작 설정

### 📈 User Activity Logging
- 음성 채널 이용 시간, 채널명, 입퇴장 시각 DB 기록
- 텍스트 채널 채팅 로그 수집

### 🎵 Music Playback
- YouTube 연동 실시간 음악 재생
- 기본 큐 및 컨트롤 기능 지원

# v1.3.0 패치노트

**릴리즈일**: 2026-03-09

## 새 기능

- 유저 상세 페이지가 추가되었습니다 — 채널별 사용 비율, 일별 음성 시간, 마이크 사용률, 입퇴장 기록을 한눈에 볼 수 있습니다
- 대시보드에서 닉네임이나 ID로 유저를 검색할 수 있습니다
- 음성 통계에 채널이 어떤 카테고리에 속해 있는지 표시됩니다
- 유저 랭킹에서 유저를 클릭하면 상세 페이지로 바로 이동합니다

## 개선

- 유저 랭킹에 프로필 사진과 순위 표시가 추가되었습니다
- AI 음성 분석이 더 안정적으로 동작합니다
- 코드 품질 개선 및 잠재적 오류 수정

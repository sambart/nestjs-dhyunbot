# v1.3.0 패치노트

**릴리즈일**: 2026-03-09

## 새 기능

- 유저 상세 페이지가 추가되었습니다 (채널별 파이차트, 일별 바차트, 마이크 사용률, 입퇴장 이력)
- 대시보드에서 유저를 닉네임/ID로 검색할 수 있습니다
- 음성 통계에 카테고리(채널 그룹) 정보가 표시됩니다
- 유저 랭킹 테이블에서 유저 클릭 시 상세 페이지로 이동합니다

## 개선

- 유저 랭킹 테이블에 아바타, 순위 표시가 추가되었습니다
- Gemini AI 분석 서비스의 타입 안전성이 개선되었습니다
- ESLint no-explicit-any, react-hooks 경고/에러가 모두 해결되었습니다

## 데이터베이스

- member 테이블에 avatarUrl 컬럼 추가
- voice_daily, channel 테이블에 categoryId/categoryName 컬럼 추가

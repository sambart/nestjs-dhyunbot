# v1.5.0 패치노트

**릴리즈일**: 2026-03-10

## 새 기능

### 봇 모니터링 대시보드
- 봇 상태를 웹 대시보드에서 실시간으로 모니터링할 수 있습니다

### 모코코 사냥 점수 기반 시스템
- 세션 수, 동석 시간, 고유 모코코 수에 가중치를 적용한 점수 시스템
- 점수별 가중치(세션당/분당/모코코당) 설정 가능
- Redis 기반 실시간 랭킹
- 일별 집계 및 리셋 주기(월간/커스텀) 설정

### 모코코 사냥 — 플레이횟수 카운팅 설정
- **플레이횟수 최소 참여시간**: 설정된 시간 미만의 세션은 플레이 횟수에 포함하지 않습니다
- **플레이횟수 시간 간격**: 직전 세션 시작으로부터 설정된 시간 이내에 시작된 세션은 동일 1회로 병합합니다
- 미션 설정의 플레이횟수 옵션과 동일한 방식으로 동작합니다

### 신입미션 관리 기능
- 웹 대시보드에서 미션 목록 조회 및 관리
- **미션 숨기기**: Embed에서 특정 미션을 숨길 수 있습니다 (hiddenFromEmbed)
- **탈퇴 상태**: 서버를 떠난 멤버의 미션이 자동으로 LEFT 상태로 변경됩니다
- **멤버 이름 저장**: 미션 생성 시 멤버의 서버 닉네임이 저장되어 탈퇴 후에도 이름이 표시됩니다

### 환영인사 커스텀 메시지
- 환영인사에 커스텀 텍스트 메시지(welcomeContent)를 설정할 수 있습니다

## 변경

### Embed 갱신 방식 개선
- 설정 저장 시 미션/모코코 Embed를 **항상 삭제 후 새로 전송**합니다
  - 기존: 설정 변경이 감지된 경우에만 재전송, 변경 없으면 기존 메시지 편집
  - 변경: 저장 시 항상 기존 메시지 삭제 → 새 메시지 전송
- Embed 갱신 실패 시 에러가 로그에 기록됩니다 (기존에는 무시됨)

### 고정메시지
- 봇 메시지 트리거 개선: 봇의 모든 메시지를 무시하던 것을 고정메시지 재전송만 무시하도록 변경
- 무한루프 버그 수정: 메시지 ID 추적 방식에서 채널 잠금 기반으로 변경 (레이스 컨디션 해결)

### Discord API 캐시 최적화
- `guilds.fetch()` → `guilds.cache.get()` 전환으로 불필요한 API 호출 제거
- `channels.fetch()` → `channels.cache` 전환

## 버그 수정

- 봇 재시작 시 음성 퇴장시간(leftAt)이 기록되지 않던 문제 해결
- `/me` 커맨드 이름 표시를 서버 닉네임으로 변경
- `/me` 프로필 카드 렌더러 수정
- 음성활동 바차트 YAxis 너비 및 동적 높이 조정
- Dockerfile.prod에 한국어/이모지 폰트 추가 (캔버스 텍스트 렌더링 문제)

# v1.10.0 패치노트

**릴리즈일**: 2026-03-15

## 새 기능

### Co-Presence 관계 분석 대시보드
- 멤버 간 동시접속(Co-Presence) 관계를 시각화하는 대시보드가 추가되었습니다
- 네트워크 그래프, 친밀도 TOP N, 일별 동시접속 추이 차트, 관계 상세 테이블, 고립 멤버 목록 제공
- 최소 임계값(분) 슬라이더로 그래프 필터링 가능

### 비활동 회원 관리
- 음성 활동 기반으로 비활동 회원을 자동 분류하고 조치할 수 있습니다
- 비활동 멤버 분류, DM 알림, 역할 변경, 강제퇴장 기능 제공
- 웹 대시보드에서 비활동 판정 기간 및 조치 설정 가능

### 음성 활동 자가진단
- `/자가진단` 커맨드가 추가되었습니다
- 활동 패턴, 관계 다양성, 뱃지 달성도를 종합 분석하여 AI 기반 맞춤형 피드백을 제공합니다
- LLM 추상화 레이어를 통해 다양한 AI 모델을 지원합니다

### 관계 다양성 점수
- HHI(허핀달-허쉬만 지수) 기반 관계 다양성 점수(0~100점)가 추가되었습니다
- 다양한 멤버와 고르게 교류할수록 높은 점수를 받습니다

### 서버 개요 대시보드
- 서버의 전체 현황을 한눈에 파악할 수 있는 개요 페이지가 추가되었습니다
- 주간 음성 활동 차트, 비활성 멤버 요약, 신입 미션 현황을 확인할 수 있습니다

### 신입 관리 대시보드
- 웹 대시보드에서 신입 미션 이력과 모코코 사냥꾼 랭킹을 조회할 수 있습니다
- 사냥꾼 상세 조회: 각 사냥꾼이 도움한 모코코 목록, 동석 시간, 세션 수 확인 가능
- 미션 이력에 현재 플레이타임 및 멤버 이름이 자동으로 보충됩니다

### 모니터링 가용률 개선
- 봇이 꺼져있는 동안의 누락 구간을 OFFLINE으로 채워 정확한 가용률을 산출합니다
- 예상 데이터 포인트 대비 ONLINE 비율 방식으로 변경되어 다운타임이 정확히 반영됩니다

## 변경

### 모코코 사냥 Embed UI 개선
- Embed 템플릿이 간결하게 개선되고 모코코 리스트가 추가되었습니다
- 모코코 사냥꾼에서 봇이 제외됩니다

### 신입미션 Embed 표시 범위 확대
- Embed에 진행 중(IN_PROGRESS)뿐 아니라 완료(COMPLETED), 실패(FAILED) 미션도 표시됩니다

### 모코코 부트스트랩 고유 모코코 수 계산 개선
- 일별 집계 합산 대신 세션 기반 고유 ID 추출로 중복 카운트 문제가 해결되었습니다

### 설정 UI 개선
- 미션/모코코 설정 UI가 접이식 섹션 구조로 개선되었습니다
- 고정메시지 설정 UI가 카드 목록에서 다중 탭 패턴으로 변경되었습니다

### /me 프로필 카드 개선
- 프로필 카드 레이아웃이 개선되고 스탯이 통합되었습니다

### 대시보드 사이드바 개편
- **서버 개요**, **신입 관리** 메뉴 추가
- 유저 검색 페이지를 음성 활동 대시보드 내 유저 상세로 통합

### 비활성멤버 기간 옵션 변경
- 비활성 판정 기간 옵션이 14일 → 15일로 변경되었습니다
- 비교 분석 기본 기간도 15일로 통일되었습니다

### 자가진단 프롬프트 개선
- LLM 프롬프트 작성 지침이 구체화되어 더 실용적인 피드백을 제공합니다

### 패키지 매니저 마이그레이션
- npm에서 pnpm으로 패키지 매니저가 변경되었습니다

### 코드 품질 도구 도입
- Husky + lint-staged + commitlint 설정 추가
- ESLint type-checked rules 추가

## 버그 수정

- 관계분석 일별 동시접속 추이 차트에서 UTC/KST 불일치로 오늘 날짜가 누락되던 문제 수정
- 관계 상세 테이블의 날짜가 ISO 타임스탬프 형태로 표시되던 문제 수정
- 동시접속 추이 쿼리의 정수 나눗셈으로 소량 데이터가 0으로 표시되던 문제 수정
- 모코코 미활동 사용자의 자가진단 Embed에서 비정상 수치가 표시되던 문제 수정
- 비활성 멤버 분류 시 Discord Gateway rate limit 에러 처리 추가
- co-presence 세션 데이터가 DB에 저장되지 않던 문제 수정
- 음성 채널 alone 판정에서 봇이 제외되지 않던 문제 수정
- 자동방 채널명 넘버링이 카테고리를 무시하고 전역으로 적용되던 문제 수정
- 환영인사 메시지본문(welcomeContent) 저장 누락 수정
- 신입미션 Embed 중복 사용자 표시 및 관리탭 불일치 수정
- 신입미션 Embed 미생성 버그 수정
- MocoResetScheduler 순환 참조 해결
- MissionStatus.LEFT 누락으로 인한 빌드 에러 수정
- DEFAULT_CONFIG에 welcomeContent 누락으로 인한 웹 빌드 에러 수정
- CI/Docker 환경에서 husky not found 오류 수정
- deploy workflow에 pnpm setup 누락 수정

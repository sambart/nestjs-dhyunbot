# v1.4.0 패치노트

**릴리즈일**: 2026-03-10

## 새 기능

- `/me` 커맨드가 추가되었습니다 — 이미지로 개인 음성 프로필 카드를 확인할 수 있습니다
  - 최근 15일 음성 활동 요약 (총 시간, 활동일, 일평균)
  - 마이크 ON/OFF 통계 및 사용률
  - 혼자 있던 시간, 피크 요일, 주 평균
  - 15일 일별 활동 차트
  - 서버 내 순위 표시

## 변경

- `/voice-time`, `/voice-rank` 커맨드가 `/me`로 통합되어 삭제되었습니다
- `/my-voice-stats`, `/voice-leaderboard`는 기존과 동일하게 유지됩니다

## 기타

- 한국어 및 이모지 폰트 지원 추가
- 대시보드와 동일한 색감 적용
